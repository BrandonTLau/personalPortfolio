School Project: Polynomial Calculator

Project Overview

This project involved creating a Polynomial Calculator in C++. The main goal was to develop a class that can handle polynomial expressions and perform various operations on them.

Features

  • Operations: Addition, Subtraction, Multiplication, Division
  • Dynamic Memory Management: Used for storing polynomial coefficients
  • Constructor: Initializes polynomials with given coefficients

Technical Details

The Polynomial Calculator was implemented using C++. The main components of the project include:

  • A Polynomial class with methods for each mathematical operation
  • Dynamic arrays to manage polynomial terms
  • Custom exception handling for invalid operations
