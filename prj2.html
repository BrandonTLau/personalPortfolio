School Project: Passenger Airlines Seating Chart

Project Overview

This project was developed during my internship and involves creating a system to manage passenger seating assignments on an airplane. The goal was to design a flexible and efficient seating chart system that could adapt to various seating configurations.

Features

  • Dynamic Seating Management: Adjust seating arrangements based on user input.
  • Flexible Input Options: Seating assignment via console or file reading.
  • Visualization: Provides a text-based visualization of seating assignments.

Technical Details

The Passenger Airlines Seating Chart system was implemented using C++. The project includes the following components:

  • Dynamic arrays to handle variable seating capacities.
  • Functions to assign and display seating based on user preferences.
  • File I/O for importing and exporting seating arrangements.
